In the PyBank part of the challenge, we were required to create a Python script to analyse the financial records of a company. Using a dataset provided I was able to create a script that returned calculations from the financial data including: Total Months, Net Total Profit/Loss, Average Changes of Profit/Loss Over the Entire Period, the Greatest Increase in Profit and the Greatest Decrease in Losses.

In the PyPoll part of the challenge, we were required to create a Python script to analyse the votes of poll data from a small rural town. The final results included: Total Votes, Percentage of Votes for Each Candidate, Total Number of Votes for Each Candidate, and the Winner.
